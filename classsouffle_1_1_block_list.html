Detailed Description

template<class T>
class souffle::BlockList< T >

A class that is designed to mimic std::list, but with better destructor speed It achieves this by allocating comparatively large chunks of memory as it needs This is not thread safe, except, when there is guarantee of at most one writer AND the TBB data structure is in use.
